# Python-Quiz-Game

Simple command-line quiz game written in Python.

## Features
- Interactive, text-based quiz
- 5 multiple-choice / short-answer questions
- Tracks correct answers and shows percentage score
- Case-insensitive answer checking

## Requirements
- Python 3.6+

## Usage (Windows)
1. Open Command Prompt or PowerShell in the project folder.
2. Run:
```
python Quiz_Game.py
```
